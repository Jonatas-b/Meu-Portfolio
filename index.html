TaskGame

HTML · CSS · JavaScript

TaskGame é uma aplicação web mobile que transforma tarefas do dia a dia em um sistema de progresso gamificado. A proposta é simples: concluir tarefas gera XP, subir de nível cria motivação real e o feedback visual ajuda a manter constância. O foco não é só listar tarefas, mas estimular hábito e disciplina usando conceitos de jogos.
